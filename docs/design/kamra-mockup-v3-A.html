Mi változott

A kártya = valódi .meal-card

44px bal → készlet (volt: étkezés ideje)
meal-name → tétel neve (Antonio)
meal-meta → forrás · márka/timing
macros-line → P/C/F · NOVA · lejárat
meal-kcal jobb → kcal/100g · supp: dózis

Rendszer-primitívek

  • .sub-nav/.sub-tab — a Fuel fülsor 1:1.
  • .divider + színes nub — típus-szekció fej.
  • .tag (warning) — a STIM jelölő.
  • .macro-cell chamfer — a 4 stat-cella.
  • .meal-window-push — a „figyelmet kér" csík.
  • .iphone + .tabbar — valódi keret/sáv.

Típus-jelölés

Étel — semleges kártya, makró-sor
Supplement — bal kék szegély, dózis-fókusz
Stimuláns — bal rózsa szegély + STIM tag

Az egyetlen új molekula a típus-váltó (a kért hangsúlyos Mind/Étel/Supp/Stim) — de a rendszer chamfer + brand-fill nyelvén.
