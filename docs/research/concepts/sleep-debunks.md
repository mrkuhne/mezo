# Sleep Debunks — myths mezo builds nothing on

**[Matthew Walker](../entities/matthew-walker.md)** spends part of the interview naming popular sleep
"hacks" that **don't work**. This page is a **negative spec**: it records each debunk *with its
reason* so future work doesn't accidentally rebuild a myth. This is the complement to
**[QQRT](qqrt.md)** — what good sleep is *not*.

## The list (claim → why it fails)

- **Fixed 90-minute "smart-wake" devices** — real sleep cycles range **70–120 min**, so a fixed
  90-min assumption is, in his words, "nonsense." No cycle-timed alarm feature.
- **"8 hours for everyone"** — quantity is a **7–9 h range**, individual; a single hard number is wrong.
- **Blue-light-blocking as the main lever** — the real problem is **engagement** ("bed rotting" /
  doom-scrolling), **not the light wavelength**; the famous iPad study **didn't replicate**. So mezo's
  wind-down nudges target *behavior* (dim + detox), not blue-light filters.
- **Magnesium as a sleep aid** — common forms (oxide / citrate) **don't cross the blood-brain
  barrier** → "expensive urine"; only L-threonate or a true deficiency has a case. No magnesium prompt.
- **Melatonin as a daily supplement** — it is a **darkness *signal*, not a sleep generator**; only
  ~**3.4 min** faster onset; the correct dose is **0.1–3 mg** (NOT the 10–20 mg sold), and it is for
  **jet lag / circadian-phase disorders only**. No "take melatonin" habit.
- **Counting sheep** — actively makes sleep onset **worse**.
- **Ambien / Z-drugs / benzodiazepines** — "**sedation ≠ sleep**"; Ambien **cuts glymphatic clearance
  ~30–40%**. Not surfaced as a sleep solution.

## The rule this encodes

mezo deliberately builds **no features** on any item above. The one narrow, clinical exception Walker
endorses (ashwagandha + phosphatidylserine for "tired but wired"; DORAs + CBT-I as prescription-grade)
is treated as **education-only**, never an in-app supplement/drug prompt. The DEBUNK/myth cards are
explicitly held in reserve in the cluster roadmap
([cluster notes](../../superpowers/specs/2026-07-23-sleep-routine-cluster-notes.md) §4) — recorded
here so the boundary is documented, not so it becomes a feature.
